# HaalChaal Support

A separate React Native support companion app for the Laravel-based HaalChaal platform.

## Core ideas

- Payments support
- Live location and tracking
- In-app chat
- Future Laravel API/database integration

## Current state

This is a starter scaffold only. The backend connection will be added later, so the screens currently use local placeholders and stubbed services.

## Folder structure

- `App.js` - app entry point
- `src/navigation` - stack navigation
- `src/screens` - feature screens
- `src/components` - reusable UI pieces
- `src/services` - future API/service layer

## Next step

When ready, connect this app to the Laravel backend with API endpoints for authentication, messages, payment status, and live tracking data.
